# Personal Portfolio Website

Personal Portfolio Website is a responsive web application currently under development using HTML, CSS, and JavaScript. The portfolio is designed to showcase my projects, technical skills, education, and contact information in a clean and professional manner. The project focuses on responsive design, modern user interface development, and creating an engaging user experience across different devices.
## Features
- Responsive Design
- About Me Section
- Skills Section
- Projects Showcase
- Contact Form
- Smooth Navigation

## Technologies Used
- HTML
- CSS
- JavaScript
